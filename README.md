# sticky-note
A sticky note using node and express


### 安装方法

打开终端 ，输入：

```
git clone git@github.com:shenzekun/sticky-note.git
```

切换到克隆下来的文件夹：

```
cd sticky-note
```
然后输入：

```
npm install
```



### 运行方法


```
node bin/www
```

打开浏览器：

```
http://localhost:3666/
```

### 实现功能

* 第三方登录
* 添加笔记（登录成功后）
* 分权限删除笔记
* 修改笔记
* 使用 markdown
* 笔记拖拽

### 涉及到的知识点
* `sass`(css 预编译器)
* `webpack`（自动化构建工具，实现LESS,CSS,JS编译和压缩代码）
* `express` (基于 Node.js 平台的 web 开发框架)
* `html`
* `css`
* `Node.js`
* `jQuery`
* `sequelize`(Node的ORM框架Sequelize操作数据库)
* `passport`(实现第三方登录)
